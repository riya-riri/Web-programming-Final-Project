Events organized by NIJ

Generally, NIJ events are planned quarterly although some events such as Autumn welcome BBQ and Graduation Farewell Party are done yearly.

The main purpose of these events is for students to interact and socialize while enjoying food and game activities.

Since the club is run mainly by international students, English is used as the core language.
